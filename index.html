Salty Engine

Being my longest running project, Salty Engine is a completely usable 2D java game library. I myself and in cooperation with other programmers developed multiple games with it, some of which are published. The downside of the project's age is that older code is from times where I was practically a beginner.

LT3

LT3 is a simplistic, yet fully functional chess UI + a very mediocre AI. According to multiple testing against myself and friends, I would estimate it to have a strength that would place it at around 1350 Elo on chess.com.
Vast optimization is still to be done; no from of mini-max is implemented, look up tables are limited.

Escape The Junk

Escape the Junk was developed by myself and a friend of mine for the Ludum Dare jam 42.
The game was made using Salty Engine and is about a robot trying to escape a junk press. A prequel is currently in the making and will explain how the poor robot ended up in that miserable situation.
